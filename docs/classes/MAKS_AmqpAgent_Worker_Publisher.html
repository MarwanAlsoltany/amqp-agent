MAKS\AmqpAgent\Worker\Publisher

A class specialized in publishing. Implementing only the methods needed for a publisher.

Example:
```
$publisher = new Publisher();
$publisher->connect();
$publisher->queue();
$publisher->exchange();
$publisher->bind();
$publisher->publish('Some message!');
$publisher->disconnect();
```
